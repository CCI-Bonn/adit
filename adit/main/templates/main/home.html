Welcome to ADIT

ADIT (Automated DICOM Transfer) is a swiss army knife to exchange DICOM data between various systems

Selective Transfer
Search for specific studies and transfer them another DICOM server or download them to a folder (optionally pseudonymized).
Batch Transfer
Transfer multiple studies specified in a CSV file to another DICOM / PACS server or download them to a folder (optionally pseudonymized).
Continuous Transfer
Continously transfer studies to another DICOM / PACS server starting by a specified date until an end date (or infinite).
