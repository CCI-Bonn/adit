Welcome to ADIT {% include "core/_adit_logo.html" %}

ADIT (Automated DICOM Transfer) is a swiss army knife to exchange DICOM data between various systems.

Studies can be transferred between different DICOM / PACS servers or can be downloaded to a network drive. The transferred / downloaded data can be pseudonymized on the fly and can be tagged with trial information. The files can also be downloaded to an encrypted 7zip archive.

ADIT supports different modes to accomplish the transfer or download:

Selective Transfer
Search and select specific studies to transfer or download.
Batch Transfer
Transfer or download multiple studies specified in a batch file.
Continuous Transfer
Continuously transfer studies from a certain point in time (TBR).
Batch Query
Query studies on a PACS server specified in a batch file.
DICOM Explorer
Explore the DICOM data of a PACS server.
