Welcome to ADIT

ADIT (Automatic DICOM Transfer) is a service to transfer or download DICOM data with the following features:

  • Transfer DICOM data between DICOM servers
  • Download DICOM data to a network folder
  • Pseudonymize the DICOM data on the fly
  • Specify a trial name or ID for the transfered data (stored in the DICOM header)
  • Download the data to an encrpyted archive (7-Zip)
  • Batch transfer or download multiple studies by uploading an Excel file
  • Search for a single study to transfer or download using a web interface
  • Find Patient IDs using the name and birth date
  • Check if a list of studies is present on a DICOM server
